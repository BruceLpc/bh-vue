BhPagination

分页
Source:
  • bh-pagination/bhPagination.vue, line 2
事件(Dispatch):
  • page-change -页码变化时触发，参数为 (pagenum, pagesize, total)
Examples

html

    <bh-pagination :pagenum.sync='pager.pagenum' :pagesize='20' :total-size='pager.total' @page-change='gotoPage'></bh-pagination>

javascript

    export default {
        data () {
            return {
                pager: {
                    pagenum: 0,
                    total: 99
                }
            }
        },
        methods: {
            gotoPage: function(pagenum, pagesize, total) {
                console.log('go to page:' + pagenum + ', pagesize:' + pagesize + ', total:' + total);
            }
        },
        components: {BhPagination}
    }

属性

props

Properties:
Name Type Attributes Default Description
mode String <optional>
 advanced 分页模式 'advanced'/'simple'
pagesize Number <optional>
 10 一页展示的数据，必须为 pageSizeOptions 中的一项
pageSizeOptions Array <optional>
 [10, 20, 50, 100] 下拉框里面的数据
selectedIndex Number <optional>
 0 下拉框的默认的index
pagenum Number <optional>
 0 当前的页码
totalSize Number <optional>
 0 总共多少条数据，当此属性变化时会自动重新渲染组件
