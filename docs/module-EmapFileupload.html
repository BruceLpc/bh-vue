EmapFileupload

文件上传[EMAP]

事件(Dispatch):
  • add -文件新增时触发，返回data（文件信息对象）
  • submit -文件提交时触发，返回data（文件信息对象）
  • done -文件提交成功后触发，返回data（文件信息对象）
  • fail -文件提交失败时触发，返回data（文件信息对象）
