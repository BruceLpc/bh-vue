BhEditPanel

复杂面板组件，支持编辑，删除，锁定标志等常用功能
Source:

Example

html

<bh-edit-panel title='用户信息' :on-edit='onEdit' :on-save='onSave' :on-cancel='onCancel' :on-del='onDel'>

Name Type Attributes Default Description

 false 是否为编辑状态

 取消编辑时的回调函数，返回promise对象
