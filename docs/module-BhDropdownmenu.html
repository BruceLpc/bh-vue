BhDropdownmenu

简单下拉按钮组件
Source:
  • bh-dropdownmenu/bhDropdownmenu.vue, line 2
事件(Dispatch):
  • clickitem - 单击某个下拉按钮时触发
Examples

html

    <bh-dropdownmenu :source='btns' type='button' icon='clouddownload' title='批量下载' display-member='name' value-member='sid' @clickitem='openMultiDownload'></bh-dropdownmenu>

javascript

    var btns = [{name: 'allow', sid: 1}, {name: 'abc', sid: 2}, {name: 'badf', sid: 3}, {name: 'best world', sid: 4}, {name: 'car', sid: 5}, {name: 'choice', sid: 6}];
    export default {
        data: function() {
            return {
                btns: btns
            }
        },
        methods: {
            openMultiDownload: function(item) {
                console.log(item);
            }
        }
    }

属性

props

Properties:
Name Type Attributes Description
title String 外部按钮显示文字
source Array 数据源，可以为普通对象数组，此属性变化会触发重新渲染；{name: 'add', sid: 4, disable: false, url: 'http://www.baidu.com'}
displayMember String <optional>
 显示字段的名称
valueMember String <optional>
 取值字段的名称
type String <optional>
 下拉按钮类型，分button和link两种类型，为button时表示普通按钮，单击触发clickitem事件；为link时表示超链接，单击跳转到相应url
icon String <optional>
 外部按钮左侧显示的图标，默认不显示
