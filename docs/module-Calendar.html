Calendar

日历控件
Source:
  • calendar/calendar.vue, line 2
事件(Dispatch):
  • item-click -点击日期的时候触发
  • next-month -切换到下月时候触发
  • last-month -切换到上月时候触发
  • month-change -切换到其他月分时候触发
  • today -切换到当日时候触发
Examples

html

    <calendar :events='events'></calendar>

javascript

    import Calendar from 'components/calendar/calendar'
    export default {
       data: function() {
           return {
               events: [
                   {start:'2016-07-06', end:'2016-07-06', title:'CLNDR GitHub Page Finished', url:'http://github.com/kylestetz/CLNDR'},
                   {start:'2016-07-11', end:'2016-07-30', title:'CLNDR GitHub Page Finished', url:'http://github.com/kylestetz/CLNDR'}
               ]
           };
       },
       components: {Calendar}
    }
