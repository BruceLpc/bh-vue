BhDropdown

简单下拉列表组件
Source:
  • bh-dropdown/bhDropdown.vue, line 2
事件(Dispatch):
  • change - 选择项发生变化的时候触发，参数为数据对象
Examples

html

    <bh-dropdown :source='words' :current.sync='dditem' display-member='name' value-member='sid' @change='onselect'></bh-dropdown>

javascript

    var words = [{name: 'allow', sid: 1}, {name: 'abc', sid: 2}, {name: 'badf', sid: 3}, {name: 'best world', sid: 4}, {name: 'car', sid: 5}, {name: 'choice', sid: 6}];
    export default {
        data: function() {
            return {
                words: words,
                dditem: 3
            }
        },
        methods: {
            onselect: function(item) {
                console.log(item);
            }
        }
    }

props

Properties:
Name Type Attributes Default Description
current Object <optional>
 默认选择项（对象，优先级高于 selectedIndex），此属性变化会触发选择项改变
selectedIndex Number <optional>
 默认选择项（索引）
source Object 数据源，可以为普通对象数组，此属性变化会触发重新渲染
placeholder String <optional>
 请选择 提示输入文字
width Number <optional>
 宽度
height Number <optional>
 高度
displayMember String <optional>
 显示字段的名称
valueMember String <optional>
 取值字段的名称
checkable Boolean <optional>
 false 是否允许多选
checkedIndexes Array <optional>
 默认选择的索引列表[多选]
url String <optional>
 从服务器加载数据时需要设置此属性，则不需要设置source属性
