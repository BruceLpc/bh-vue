BhCheckbox

复选框

事件(Dispatch):
  • change - 选择状态发生变化时触发，参数为数据对象：{value: 1, checked: true, el: dom, extra: {a:1, b:2}}
Example
<bh-checkbox ischeck='false' :disabled='flag'>选项一</bh-checkbox>

属性

props

Properties:
Name Type Attributes Default Description
value String 绑定框
ischeck Boolean <optional>
 false 是否选中
disabled Boolean <optional>
 false 是否禁止操作
extra Object <optional>
 绑定的数据，除了name/value之外需要额外绑定的数据，可以在change事件中获取此属性
inGroup Boolean <optional>
 false 是否在checklist中，一般不需要修改此属性
